# Calculadora-Cientifica
Proyecto colaborativo desarrollado en Java que integra diversos módulos matemáticos en una sola aplicación funcional

# Equipo y Contribuciones

- Felipe Rain: Módulo de Operaciones Básicas (Aritmética)
- Sebastián Durán: Módulo de Geometría (Áreas y perímetros)
- Sebastián Colomera: Menú principal, Análisis Lineal e Integración del repositorio

# Estructura del Código

src/calculadora/
├── AnalisisLineal.java      # Ecuaciones y sistemas
├── CalculadoraApp.java      # Clase principal y Menú
├── Geometria.java           # Funciones geométricas
└── OperacionesBasicas.java  # Aritmética fundamental

# Tecnologías

- Lenguaje: Java 21
- Herramientas: Git, GitHub e IntelliJ IDEA
